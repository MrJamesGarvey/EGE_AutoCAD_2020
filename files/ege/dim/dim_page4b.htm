Feature Dimensions

 





  1. Repetitive features or dimensions may be specified by using the symbol “X” along with the number of times the feature is repeated. There is no space between the number of times the feature is repeated and the “X” symbol, however, there is a space between the symbol “X” and the dimension (i.e. 8X n10).

 

  1. Equally spaced features are specified by giving the number of spaces followed by the repeated feature symbol “X”, a space, and then the dimension value. One space may be dimensioned and given as a reference value.
