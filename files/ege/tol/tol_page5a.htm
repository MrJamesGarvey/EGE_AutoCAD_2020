Inch Tolerances

 





Types of Fit

 

There are four major types of fits.

 

  1. Clearance Fit
  2. Interference Fit
  3. Transition Fit
  4. Line Fit

 

What is a clearance fit?

 

  • There is always a space.
  • Min. Clearance > 0

 

What is an interference fit?

 

  • There is never a space.
  • Max. Clearance <= 0

 

What is a transition fit?

 

  • Depending on the sizes of the shaft and hole there could be a space or no space.
  • Max. Clearance > 0
  • Min. Clearance < 0

 

What is a line fit?

 

  • There is a space or a contact (hole dia = shaft dia)
  • Max. Clearance > 0
  • Min. Clearance = 0
